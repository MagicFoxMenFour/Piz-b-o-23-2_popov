# **Теоретическая часть**

## **1. Представления графов**

### **Матрица смежности**

Таблица размера V×V, где V — количество вершин.

* `matrix[u][v] = weight` — ребро существует
* `0` — ребра нет

**Память:** O(V²)
**Плюсы:** моментальная проверка ребра O(1)
**Минусы:** очень большой расход памяти в разреженных графах

---

### **Список смежности**

Хранит только реальные связи между вершинами.

**Память:** O(V + E)
**Плюсы:** экономия памяти, быстрый обход соседей
**Минусы:** проверка ребра — O(deg(v))

---

## **2. Основные алгоритмы**

### **2.1. BFS ― поиск в ширину**

* Использует очередь
* Обходит граф слоями
* Находит кратчайший путь в невзвешенном графе

**Сложность:** O(V + E)

---

### **2.2. DFS ― поиск в глубину**

* Обходит граф по одному пути, делая возвраты
* Есть рекурсивная и итеративная реализация

**Сложность:** O(V + E)
**Применения:** топосорт, поиск циклов, компоненты связности

---

### **2.3. Алгоритм Дейкстры**

* Находит кратчайшие пути в графе с неотрицательными весами
* Использует приоритетную очередь

**Сложность:** O((V + E) log V)
**Ограничение:** отрицательные веса недопустимы

---

### **2.4. Топологическая сортировка**

Используется для DAG-графов.

**Сложность:** O(V + E)

---

### **2.5. Компоненты связности**

Определяют группы вершин, между которыми есть путь.

**Сложность:** O(V + E)

# **Анализ результатов**

* BFS корректно находит порядок обхода и минимальные пути в невзвешенных графах.
* DFS (оба варианта) дают одинаковый порядок для данного графа.
* Алгоритм Дейкстры работает корректно при неотрицательных весах.
* Поиск компонент связности выделяет все группы вершин.
* Топологическая сортировка обнаруживает циклы, если они есть.
* Эксперименты с производительностью подтверждают теоретическую асимптотику:

  * матрица смежности неэффективна для больших разреженных графов;
  * список смежности оптимален при E << V².
